# quant-research-lab

這個 repo 用來保存可攜式研究脈絡，重點是：

- TODO 清單
- 決策紀錄（我們討論後拍板的方向）
- 重要里程碑與驗收結論

## 不放的內容

- 大型實驗資料（raw data、cache、logs）
- 測試用程式碼與臨時腳本
- 可重建的產物（例如 backtest 中間檔）

## 建議結構

- `TODO.md`：目前待辦與狀態
- `decisions/`：重要決策（架構、工具、規範）
- `handover/`：環境前移或交接所需摘要

## 原則

1. 可攜性優先：新環境 clone 後可快速延續工作
2. 精簡優先：只存「必要上下文」
3. 可追溯：每次重大決策都落檔
