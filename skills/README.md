# skills

這裡放「專案自定義 skills」的可攜版本，作為研究與流程規範的單一真源。

## 已納管
- `workflow-consistency/`
- `artifact-lifecycle-manager/`

## 不直接收錄
- 外部第三方 skills 整包（例如 `python-skills`）

外部 skills 以來源連結與安裝指令管理，避免 vendor 大量內容造成維護負擔。

建議安裝方式：
- 來源：`https://github.com/awwesomeman/python-skills.git`
- 使用其 `install.sh` 安裝到 OpenClaw
